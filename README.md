## 주요 기능
<b>메인화면</b>
  - 스토어홈, 베스트, 타임세일, 특가 상품 4개 메뉴로 나누고 메뉴에 맞는 물품들을 나열
  - 네이버 로그인 API로 편리한 로그인
  - Swiper 기능을 이용하여 페이지에 편리성 증대
    
<b>검색</b>
  - 최근 검색어, 인기 검색어들을 볼 수 있는 화면과, 자신의 원하는 물품을 찾아 나열

<b>장바구니</b>
  - 자신의 구입한 물품들을 나열 배송 상태, 상품 정보, 상품 후기 작성란을 포함해 나열
    
<b>마이페이지</b>
  - 자신의 정보 변경, 주소입력, 최대 10개의 찜목록을 나열
    
<b>찜 목록</b>
  -  상품중에 자신의 원하고 관심 있는 선택한 물품들을 나열
    
<b>관리자</b>
  - 관리자 권한만 있는 사람만 게시글 삭제, 추가, 회원관리, 배송 상태를 변경 가능(직접 배송처리가 불가능하기때문에 관리자가 배송 완료를 누르면 배송 완료처리)

<b>상품페이지</b>
  - 상품의 기본적인 정보들과 후기, 찜, 공동구매 창
